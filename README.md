# Desktop Application for a Gym Company

This desktop application was developed to manage the company's customers, income and expenses.

foto

## 🛠️ Installation
```
npm install
```

## 🌱 On Development

To run in the browser:
```
npm run serve
```

To run in the desktop:
```
npm run electron:serve
```

## 💻 On Production
```
npm run build
```
